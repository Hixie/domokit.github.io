void takeChild( )

Detaches the single child of this object from this object, without calling remove() on that child. Only called when singleChild returns a non-null node. Override this if you override singleChild to return non-null.

Source

void takeChild() {
  assert(singleChild == child);
  assert(child != null);
  _child = null;
}
