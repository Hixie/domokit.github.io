bool modal

If modal is true, a hidden layer is inserted in the widget tree that catches all touches to widgets created by routes below this one, even if this one is transparent.

If modal is false, then earlier routes can be interacted with, including causing new routes to be pushed and/or this route (and maybe others) to be popped.

ephemeral must be true if modal is false. hasContent must be true if modal is true.
