void detachChildren( )

Override this function in subclasses with child to call detach() for each child. Do not call directly.

Source

void detachChildren() {
  for (ScaffoldSlots slot in ScaffoldSlots.values) {
    RenderBox box = _slots[slot];
    if (box != null)
      box.detach();
  }
}
