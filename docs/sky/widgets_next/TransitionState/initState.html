void initState(
BuildContext context
)

Called when this object is inserted into the tree. Override this function to perform initialization that depends on the location at which this object was inserted into the tree or on the widget configuration object.

If you override this, make sure your method starts with a call to super.initState(context).

void initState(BuildContext context) {
  super.initState(context);
  config.performance.addListener(_performanceChanged);
}
