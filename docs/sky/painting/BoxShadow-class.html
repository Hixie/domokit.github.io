A shadow cast by a box

Note: BoxShadow can cast non-rectangular shadows if the box is non-rectangular (e.g., has a border radius or a circular shape).

Static Methods

lerp(BoxShadow a, BoxShadow b, double t) → BoxShadow

Linearly interpolate between two box shadows /// /// If either box shadow is null, this function linearly interpolates from a /// a box shadow that matches the other box shadow in color but has a zero /// offset and a zero blur.

lerpList(List<BoxShadow> a, List<BoxShadow> b, double t) → List<BoxShadow>

Linearly interpolate between two lists of box shadows /// /// If the lists differ in length, excess items are lerped with null.
