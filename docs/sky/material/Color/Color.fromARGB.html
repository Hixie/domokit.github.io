const Color.fromARGB(int a, int r, int g, int b)

Construct a color from the lower 8 bits of four integers.

Source 

const Color.fromARGB(int a, int r, int g, int b) :
  _value = ((((a & 0xff) << 24) |
              ((r & 0xff) << 16) |
              ((g & 0xff) << 8) |
              ((b & 0xff) << 0)) & 0xFFFFFFFF);
