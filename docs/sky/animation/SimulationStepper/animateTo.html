animateTo(double target, {Duration duration, Curve curve: Curves.linear})

Animate value of the timeline to the given target over the given duration

Returns a future that resolves when the timeline stops animating, typically when the timeline arives at the target value.

Source 

Future animateTo(double target, { Duration duration, Curve curve: Curves.linear }) {
  assert(duration > Duration.ZERO);
  assert(!isAnimating);
  return _start(new _TweenSimulation(value, target, duration, curve));
}
