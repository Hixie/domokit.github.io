bool hitTestChildren(HitTestResult result, {Point position})

Override this function to check whether any children are located at the given position

Typically children should be hit tested in reverse paint order so that hit tests at locations where children overlap hit the child that is visually "on top" (i.e., paints later).

Source 

bool hitTestChildren(HitTestResult result, { Point position }) {
  if (isVertical)
    return defaultHitTestChildren(result, position: position + new Offset(0.0, -startOffset));
  else
    return defaultHitTestChildren(result, position: position + new Offset(-startOffset, 0.0));
}
